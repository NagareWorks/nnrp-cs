# Nnrp.NativeBridge

Nnrp.NativeBridge provides Unity-compatible managed glue for native NNRP transport bridges.

Use this package when you want the managed client surface plus packaged native bridge assets for the current preview transport path.

This package depends on Nnrp.Core, Nnrp.Client, and Nnrp.Transport.Tcp, and may include runtime-specific native binaries when they are present during packing.

Install:

```powershell
dotnet add package Nnrp.NativeBridge --version 1.0.0-preview.2
```

Repository and full SDK documentation:

- https://github.com/NagareWorks/nnrp-cs
