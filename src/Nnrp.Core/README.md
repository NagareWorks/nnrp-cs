# Nnrp.Core

Nnrp.Core provides the core binary wire primitives for the current NNRP/1 contract.

Use this package when you need direct access to frame headers, metadata codecs, payload descriptors, protocol state helpers, or transport-neutral framed message types.

Typical consumers:

- protocol tooling and conformance tests
- transport adapters
- advanced client or server integrations that need low-level wire access

Install:

```powershell
dotnet add package Nnrp.Core --version 1.0.0-preview.2
```

Repository and full SDK documentation:

- https://github.com/NagareWorks/nnrp-cs
