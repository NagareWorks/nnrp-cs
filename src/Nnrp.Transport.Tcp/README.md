# Nnrp.Transport.Tcp

Nnrp.Transport.Tcp provides the TCP transport adapter for framed NNRP messages.

Use this package when you need a managed TCP transport implementation that works with the NNRP core framing model and higher-level client helpers.

This package depends on Nnrp.Core.

Install:

```powershell
dotnet add package Nnrp.Transport.Tcp --version <published-version>
```
